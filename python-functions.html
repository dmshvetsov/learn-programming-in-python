How to make functions and how to use them in Python

A function is 1 or more lines of code that we group together and give it a name.

How to make them

Let's create a simple function


                        def name_of_the_function:
                            pass

Here I using pass keyword because my function does nothing but it should contain a body.

Why we need functions

If in a program we need to repeatedly write 1 or more lines of code several times we can create a functions with these lines and reduce headache of writing and editing these lines multiple time in the program.

Reduce Repetitive Code

Can you spot repetition?


                        group_a = [23, 100, 32, 1, 2, 89]
                        sum = 0
                        for num in group_a:
                            sum += num
                        mean = sum / len(group_a)
                        print(mean) # => 41.166666666666664

                        sum = 0
                        group_b = [34, 25, 21, 80, 67, 3]
                        for num in group_b:
                            sum += num
                        mean = sum / len(group_a)
                        print(mean) # => 38.333333333333336

We can rewrite it to:


                        def mean(numbers):
                            sum = 0
                            for num in numbers:
                                sum += num
                            return sum / len(numbers)

                        group_a = [23, 100, 32, 1, 2, 89]
                        print(mean(group_a)) # => 41.166666666666664
                        group_b = [34, 25, 21, 80, 67, 3]
                        print(mean(group_b)) # => 38.333333333333336

As a result we have less lines and more readable code.

Readability of the code

What I mean by readable?

I mean that the later code with the mean function is literally easier to read.

Because of the name of the function brings some information, it is easier to read code naturally as a story.

Take a look:

How to Call Python Functions


                        def say(person_name, message):
                            print(person_name + ":", message)

Default value for argument


                        def say(person_name, message="..."):
                            print(person_name + ":", message)

Call function with positioned arguments


                        def say(person_name, message="..."):
                            print(person_name + ":", message)
                        
                        say('Dmitry', 'Here I\'m using positioned arguments')
                        say(person_name='Dmitry', message='Here I\'m using key arguments')
                        say(message='and I can change the order when using key arguments')

Call function with key arguments


                        def say(person_name, message="..."):
                            print(person_name + ":", message)
                        
                        say(person_name="Dmitry", message="here i'm using key arguments")
                        say(message="and i can change the order when using key arguments", person_name="Dmitry")

Combining positioned and key arguments


                        def say(person_name, message="..."):
                            print(person_name + ":", message)
                        
                        say(Dmitry", message="here i'm using key arguments")

One condition here we should use key arguments must follow the positional arguments.

Wrong order of positional and key arguments


                        def say(person_name, message="..."):
                            print(person_name + ":", message)
                        
                        say(Dmitry", message="here i'm using key arguments") # => SyntaxError: non-keyword arg after keyword arg
